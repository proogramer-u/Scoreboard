# Basketball Scoreboard

Features

- **Score Tracking**
  - Separate score displays for Home and Away teams
  - Add points: +1, +2, +3
  - Subtract points: -1, -2, -3
  - Reset button for each team

- **Timer Functionality**
  - 10-minute countdown timer
  - Start, Stop, and Reset controls
  - Displays time in MM:SS format

- **Visual Design**
  - Basketball-themed background
  - Custom "Cursed Timer" font for score and timer displays
  - Clean, modern UI with hover effects

## How to Use

1. **Open the application**
   - Simply open `index.html` in any modern web browser
   - No installation or server required

2. **Score Management**
   - Use the **+1**, **+2**, or **+3** buttons to add points to a team
   - Use the **-1**, **-2**, or **-3** buttons to subtract points
   - Click **Reset** to reset a team's score to 0

3. **Timer Controls**
   - Click **Start** to begin the countdown
   - Click **Stop** to pause the timer
   - Click **Reset** to reset the timer back to 10:00

Works on all modern web browsers including:
- Chrome
- Firefox
- Safari
- Edge
